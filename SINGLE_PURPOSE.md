# Single Purpose Statement - Final Version

Use **exactly** these statements on Chrome Developer Dashboard:

---

## Option 1: Short & Simple (Recommended)

**Single purpose:**
```
Edit Markdown text in the browser with live preview
```

---

## Option 2: More Detailed

**Single purpose:**
```
Write and preview Markdown content in a split-pane editor with copy-to-clipboard functionality
```

---

## Option 3: Most Detailed

**Single purpose:**
```
A browser-based Markdown editor that provides real-time preview, text formatting tools, and the ability to copy formatted content to clipboard
```

---

## Why These Work

| Element | Explanation |
|---------|-------------|
| "Edit Markdown text" | Clear, specific functionality |
| "in the browser" | Where it runs |
| "live preview" | Key feature |
| "split-pane" | UI layout |
| "copy-to-clipboard" | Export method |

---

## What to AVOID

❌ "Productivity tool" - Too vague
❌ "Markdown editor with many features" - Too broad
❌ "Helps users write better" - Subjective
❌ "Multiple functions including..." - Not single purpose

---

## Quick Copy

```
Edit Markdown text in the browser with live preview
```

---

## Dashboard Steps

1. Go to **Privacy practices** tab
2. Find **Single purpose** field
3. Paste one of the options above
4. Save
