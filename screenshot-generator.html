📸 Chrome Web Store Screenshot Generator

📋 How to Create Screenshots

  1. Option A - Use this tool: Click "Open Fullscreen Editor" below, then use browser's screenshot tool
  2. Option B - Manual: Load the extension, open fullscreen mode, and capture screenshots manually
  3. Save as PNG format with the required dimensions
  4. Upload to Chrome Developer Dashboard

💡 Requirements

  • Small tile: 640 x 400 pixels (shown in store listing)
  • Large tile: 1280 x 800 pixels (shown when expanded)
  • Minimum 1 screenshot required, up to 10 allowed
  • Use PNG or JPG format
