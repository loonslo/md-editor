图标文件说明

需要以下尺寸的PNG图标文件：
- icon16.png  (16x16像素)
- icon48.png  (48x48像素)
- icon128.png (128x128像素)

建议：
1. 使用SVG格式创建图标后转换为PNG
2. 图标应表示Markdown编辑器的主题
3. 可以使用简单的文本编辑器图标或Markdown符号
4. 确保图标在Chrome工具栏中清晰可见

可以在以下在线工具生成PNG图标：
- https://romannurik.github.io/AndroidAssetStudio/
- https://www.favicon-generator.org/
