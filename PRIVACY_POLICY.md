# Privacy Policy

## Markdown Editor Chrome Extension

**Last updated:** January 2025

### Data Collection
This extension does **not** collect, transmit, or store any personal data.

### Permissions Used
- `storage`: Used only to save your markdown content locally on your device for auto-recovery
- `clipboardWrite`: Used only when you explicitly click "Copy" buttons to copy content to your clipboard

### Data Storage
- All data is stored locally on your device
- No data is transmitted to any external servers
- You can clear your data by clicking the "Clear" button or uninstalling the extension

### Third-Party Libraries
This extension uses the following third-party libraries:
- `marked.js` - Markdown parsing (loaded locally)
- `highlight.js` - Code syntax highlighting (loaded locally)
- `DOMPurify` - HTML sanitization (loaded locally)

These libraries are bundled with the extension and do not transmit any data.

### Contact
If you have any questions about this privacy policy, please contact us through the Chrome Web Store listing.
