## 🎯 功能特性

### 1. 自动语言检测
- 读取浏览器 `navigator.language`
- 匹配最佳语言资源
- 兜底到英文

### 2. 动态文本更新
```javascript
I18N.updateUITexts(); // 一键更新所有 UI
```

### 3. 欢迎文本本地化
```javascript
I18N.getWelcomeText(); // 根据当前语言生成
```

### 4. 存储管理
- 语言偏好保存到 `chrome.storage.local`
- 跨页面同步语言设置
- 启动时自动加载
